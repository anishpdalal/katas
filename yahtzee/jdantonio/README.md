# Yahtzee

Yahtzee kata from http://codingdojo.org/cgi-bin/index.pl?KataYahtzee, done using TDD in JavaScript.

## Setup

The JavaScript code uses [grunt](http://gruntjs.com/) as the task runner, [eslint](http://eslint.org/) for static analysis, and [nodeunit](https://github.com/caolan/nodeunit) for the tests.

To install all the dependencies:

```
$ npm install --show-dev
```

To run static analysis:

```
$ grunt lint
```

To run the tests:

```
$ grunt test
```
